PARA EL ARBOL MORSE 

Tome agregando una letra a su árbol, por ejemplo: 
1.cadena . 
2. Comience en el nodo raíz 
3. Para cada carácter en la Cadena: 
4. Si es un punto, vaya a la izquierda. Si no existe ningún nodo izquierdo, agréguelo para que pueda. 
5. Si es un guión, ve a la derecha. Si no existe un nodo correcto, agréguelo para que pueda. 
6. Si tampoco lo es, lanza una Excepción ("¿cadena Morse mala"?). Esto es algo muy importante para recordar: cualquier persona puede escribir código que funcione cuando las cosas están bien; Los buenos programadores escriben un código que funciona cuando no lo están . 
7. Si hay más caracteres, vuelva al Paso 3. 
8. Establezca el carácter para el Nodo final al que llegó al carácter ASCII que se le proporcionó. 
Tenga en cuenta que al describir el proceso no he utilizado ningún Java.
